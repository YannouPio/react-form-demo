#  可折叠的多步骤表单向导
题目要求
实现一个多步骤表单向导组件，具有以下功能：
1. 表单分为多个步骤（至少3个步骤），每个步骤包含不同的表单字段
2. 用户可以自由折叠/展开任意步骤，但只能编辑当前活动步骤
3.步骤之间有验证逻辑，只有当前步骤验证通过 能进入下一步
4. 显示当前进度状态（如"歩骤1/3"）
5. 提供“上一步“和”下一步“导航按钮
6.所有步骤验证通过后显示提交按鈕
7.使用React Hook 实现
